Version Control adalah sistem yang digunakan untuk mencatat setiap perubahan pada source code secara terstruktur. Dengan version control, pengembang dapat:
- Melacak perubahan kode
- Bekerja secara kolaboratif

Git merupakan salah satu sistem version control terdistribusi yang paling populer dan banyak digunakan dalam pengembangan perangkat lunak modern.
