## ğŸ“˜ Konsep Database Server
Database server adalah sistem yang bertugas untuk menyimpan, mengelola, dan menyediakan akses data secara terstruktur dan terpusat. Database server memungkinkan banyak aplikasi atau pengguna untuk mengakses data secara bersamaan dengan tetap menjaga konsistensi dan keamanan data.

---

## ğŸ” SQL vs NoSQL
Pada Jobsheet ini dibahas perbandingan antara:
- **SQL (Relational Database)**  
  Menggunakan tabel dengan skema tetap, relasi antar tabel, dan bahasa SQL.
- **NoSQL (MongoDB)**  
  Menggunakan dokumen berbasis JSON/BSON yang fleksibel dan mudah diskalakan.

MongoDB dipilih karena mendukung penyimpanan data dalam bentuk dokumen dan sangat cocok untuk aplikasi modern dengan data yang dinamis.

---

## ğŸ§© Konsep MongoDB
MongoDB merupakan database NoSQL berbasis dokumen yang menyimpan data dalam format **JSON-like**. Struktur utama MongoDB meliputi:
- Database
- Collection
- Document
- ObjectId sebagai identitas unik data

MongoDB memungkinkan operasi **CRUD** (Create, Read, Update, Delete) secara efisien.
